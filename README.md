PYTHON-BASED BIOMETRIC FACIAL VERIFICATION SYSTEM 

To protect systems or database, the program that works with face detection and recognition algorithms to unlock system by verification of the authorized user's face is created.

Features:
-Biometric Authentication
-Data Privacy via Pickle
-Logging History

Requirements and libraries:
-Python 3.10+
-OpenCV
-Face_Recognition
-Pickle

To add authorized user:
-Faces should be added to the system one by one.
-Run "register.py" and place the face to the center as well-visible.
-To register the face, press "s".
-To exit, press "q".

To verify and unlock the system:
-Run "main.py"
-Place the face to the center as well-visible.
